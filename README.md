# sebastiandoyle.github.io

Personal website and central hub for iOS app privacy policies. Hosts privacy policy pages for 25+ published App Store apps alongside a portfolio landing page.

## What's Here

- **Landing page** — personal site with project links and bio
- **Privacy policies** — individual policy pages for each iOS app, linked from App Store listings
- **App support pages** — contact information for app users

## Apps with Privacy Policies

Each app has its policy at `/[appname]-privacy/privacy-policy.html`, including:
- CleanSnap, ZenDot, FlipCoin, Roll the Dice, Imposter, HydroMind, PocketBreath, and 20+ more

## Tech Stack

- Static HTML/CSS
- GitHub Pages hosting
- Custom domain support

## Local Development

```bash
git clone https://github.com/sebastiandoyle/sebastiandoyle.github.io.git
cd sebastiandoyle.github.io
open index.html
```
