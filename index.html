The pipeline scrapes and parses full debate transcripts, tokenises each candidate's turns, and runs sentiment and aggression scoring using scikit-learn classifiers. The output is a dataset that tracks rhetorical style across decades, showing exactly when American political debate shifted from policy substance to personal attack. Built during my economics degree as a way to apply ML to something I actually cared about.

The data tells the story. The models just make it legible.

The beauty is in the design of the functions themselves. Each one encodes a principle from negotiation theory: BATNA awareness, interest mapping, creative option generation, objective criteria. The LLM decides which to call based on what the user just said. It's not a chatbot with a system prompt. It's a program where the control flow is determined by a language model, and each branch runs real logic.

Built from three years of studying negotiation in law school. The AI handles the conversation. The functions handle the scoring.
